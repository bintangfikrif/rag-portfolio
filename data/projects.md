# Project: AirBeats
AirBeats is an interactive rhythm game developed in Python using the Pygame library. The core innovation of AirBeats is its use of computer vision and hand-tracking via OpenCV to detect player finger movements in real-time. 

# Project: Drowsiness Detection System
The Drowsiness Detection System is a final research project focused on analyzing EEG and EOG biometric signals. This system utilizes a Mamba neural network architecture to classify human alertness states. The data pipeline involves subject-wise normalization preprocessing.

# Project: Neoface
Neoface is An end-to-end Face Recognition system built with PyTorch. The project implements and compares Deep Learning architectures, specifically InceptionResnetV1 and ResNet18. It features a complete pipeline including automated face cropping, custom data augmentation, and robust model evaluation using K-Fold Cross Validation.

# Project: Deforestation Forecasting: ARIMAX vs GRU
Deforestation Forecasting dashboard provides a forecast for Indonesian deforestation by comparing advanced time-series models. Our results demonstrate that the GRU deep learning model delivers significantly more accurate predictions than the traditional ARIMAX statistical method for this complex environmental data.

# Project: Social Ads Analysis
Social Ads Analysis is a project that Engineered a BigQuery and Looker Studio pipeline to analyze 340K impressions across 40+ campaigns. Leveraged advanced SQL (array unnesting, window functions) to map conversion funnels and prove that while Finance and Health drive volume, specific Female demographics dominate actual purchase intent.

# Real-Time rPPG and Respiration Rate Tracker
Real-Time rPPG and Respiration Rate Tracker is a real-time health tracker that uses a webcam to measure heart rate and respiration rate without any physical contact. It analyzes subtle color changes on the forehead to estimate heart rate (rPPG) and tracks shoulder movements with MediaPipe to determine breathing rate. The Python application displays the live video, physiological signals, and calculated vitals in a user-friendly interface

# Visualization of E-Waste Distribution
Visualization of E-Waste Distribution is an interactive Tableau dashboard and informational poster analyzing 2022 global e-waste distribution and 2030 recycling projections. The visual narrative highlights that high-income nations drive the most e-waste per capita across device categories, while Asia leads in total volume with 30,149 million kg.

# AI-based classification of Indonesian Regional Foods
AI-based classification of Indonesian Regional Foods is an end-to-end Deep Learning pipeline in PyTorch to classify 5 classes of Indonesian traditional cuisines. Fine-tuned state-of-the-art Convolutional Neural Networks (EfficientNet-V2-S, B4, B3). Successfully achieved a peak validation accuracy of 98.20% with the EfficientNet-B4 architecture.

# Dukun vs Demit Game
Dukun vs Demit (DvD) is a tower defense game developed in Python using the Pygame library. The game's structure is object-oriented, with classes for game elements like Dukun (the player character), Demit (the enemies), and Pendopo (the base to be defended). Key gameplay mechanics include aiming and shooting projectiles, managing resources like money and health, and progressing through increasingly difficult levels.
